Google App Engine Launcher Help

Running the demo application

Google App Engine Launcher includes a fully functional demo application that you can create, run, and modify.

To create the demo application:

  • Select the menu option Help > Demos > guestbook. The application files are created in a directory named "guestbook" in your My Documents directory, and the project is added to the project list.

To run the demo application using the local app server:

  1. With the guestbook application selected in the project list, click the "Run" button (or select Control > Run; or press Control-R). The local app server starts, and the green "running" icon appears next to the project in the project list.

  2. Click the "Browse" button (or select Control < Browse; or press Control-B).

    A web browser window opens with the demo application's home page.

    Note: If the browser reports an error when it tries to connect to the server for the first time, wait a few seconds and try again.

  3. Test the application: In the browser window, click in the text box, type some text, then click the "Sign Guestbook" button.

    The application responds by adding the message to the page.

To modify the demo application:

  1. In the Launcher window, click the "Edit" button (or select Edit > Open in External Editor..., then select a text editor application; or press Control-E).

    A text editor opens with the files for the project.

  2. In the text editor, open the file named "guestbook.py". Find the line containing "An anonymous person wrote:". Change the phrase to "A very nice person wrote:", or anything else you want.

  3. If the local app server is not running for this project (the green icon does not appear to the left of the project in the project list), click the Run button.

    Tip: You don't have to restart the local app server for most changes to application source files. If the server is still running from a previous step, you can leave it running.

  4. Select the browser window, then reload the page. Previously entered messages are displayed, but with the new phrase ("A very nice person wrote:").
