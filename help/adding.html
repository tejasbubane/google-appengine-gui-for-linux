Google App Engine Launcher Help

Adding an Existing Project

If you already have a Google App Engine project, you can add it to the Launcher to run the local app server, and to deploy it to App Engine.

To add an existing Google App Engine project to the launcher:

  1. Select the menu option File > Add Existing Application...

    Or: Press Control-Shift-N.

    The "Add Existing Application" dialog opens.

  2. Click the "Browse..." button and navigate to the existing application's root directory. The root directory is the directory containing the application's app.yaml file.

  3. (Optional) Change the port number to use when running the local app server for this project. By default, a port number is selected that doesn't conflict with any other project in the Launcher.

  4. Click the "Add" button. The project is added to the Launcher project list.
