Changing Local App Server Settings

Each project can have its own settings to use with the local app server.

To change the local app server settings for a project:

  1. Select the project from the project list.

  2. Select the menu option Edit > Application Settings...

    Or: Press Control-I.

    The application settings dialog opens.

  3. Change settings as appropriate. For example, you can change the port the app server uses, or change the options used with the app server when it launches.

    Note: You can't change the application name or the application directory after the project has been created.

  4. Click the "Update" button to save the new settings.
