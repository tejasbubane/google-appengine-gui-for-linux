Google App Engine Launcher Help

Accessing the App Engine Admin Console

You can manage and monitor applications running on Google App Engine using the Admin Console from your web browser. You can open the Admin Console directly from the Launcher.

To open the Google App Engine Admin Console for a project:

  • With the project selected in the project list, click the "Dashboard" button.

    Or: Select the menu option Control > Dashboard.

    Or: Press Control-Shift-D.

    A web browser window opens with the Google App Engine Admin Console for the selected project. You may be prompted to sign in with your Google account.
